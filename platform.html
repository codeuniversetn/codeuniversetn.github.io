PCR

Sentosa SX101

A flexible automated pipetting system with minimum footprint that offers multipurpose utility for sample lysis, nucleic acid extraction and PCR setup as well as for NGS library prep. The open channel capability with a modular instrument deck also allows customized applications and primary tube handling. The unprecedented system flexibility generates a high degree of laboratory efficiency for the user. The Sentosa system comes with a complete set of extraction reagents and more than 60 different applications.

Rotor-Gene Q

Using an innovative rotary design, the Rotor-Gene Q MDx 5plex HRM Platform allows for unmatched thermal performance in PCR cycling. The use of a rotating disc ensures that every single tube is spun in a chamber of uniform temperature, using air as the conducting element. This allows for rapid thermal cycling and precise results in all tubes.

Sentosa SA Reporter

Sentosa SA Reporter retrieves Rotor-Gene Q run data and automatically performs result analysis. The sophisticated software generates fast, objective and reproducible data interpretation, saves hands on time and thereby provides potential for increased cost effectiveness.

TESTS

Vela Diagnostics offers a broad range of tests for the diagnosis of infectious diseases and cancer. Tests are validated for more than 20 different sample types providing a high degree of clinical versatility. Disease relevant test combinations and the ability to analyze up to four tests in parallel from one clinical sample produce comprehensive and actionable diagnostic results.

PCR WORKFLOW

Our integrated PCR workflow offers minimal hands-on time and fast turn-around time through automation and thus frees up operator time and reduces human error. By offering flexible system applications, instrument complexity gets reduced. Additionally, open-channel capability allows adaptation to lab-specific needs.
